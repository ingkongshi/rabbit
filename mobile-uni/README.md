## Project Setup

- mp-weixin 微信小程序
- mp-alipay 支付宝小程序
- mp-toutiao    抖音小程序
- h5    普通H5

### Install Dependencies
```sh
pnpm install
```

### Compile and Hot-Reload for Development


```sh
pnpm run dev:h5
```

### Compile and Minify for Production

```sh
pnpm run build:h5
```
